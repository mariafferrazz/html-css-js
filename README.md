# html-css-js
Este repositório reúne os projetos desenvolvidos ao longo do meu aprendizado em desenvolvimento web, com foco em HTML, CSS e introdução à otimização com JavaScript. Durante o curso, explorei conceitos fundamentais e práticos para construção de interfaces modernas e responsivas, aplicando boas práticas de estruturação e organização de código.

# Conteúdos abordados
Estruturação de páginas com HTML semântico;
Criação de formulários e validações básicas;
Estilização com CSS moderno;
Layouts utilizando:;
Flexbox;
CSS Grid;
Evolução de técnicas de posicionamento e responsividade;
Organização de código com foco em clean code;
Introdução à otimização de projetos com JavaScript;

 # Sobre os projetos

Os projetos presentes neste repositório representam minha evolução prática ao longo do curso, desde construções mais simples até aplicações com layouts mais estruturados e organizados.

# Objetivo

Consolidar conhecimentos em desenvolvimento front-end e construir uma base sólida para atuar como desenvolvedora web, com foco em criação de interfaces eficientes, acessíveis e bem estruturadas.
